  1. Click Design Tokens JSON tab → Download tokens.json

    3. 
              
  3. Open the plugin → click Load from file → select your tokens.json
    4. 
              
  4. Apply tokens to your Figma file — colors, typography, spacing will be synced
    5. 
            


            
🔌 Method 2 — Figma Plugin Runner

            
    
              
  1. In Figma, go to Plugins → Development → New Plugin
    2. 
              
  2. Choose Run once script
    3. 
              
  3. Paste the code from the Figma Plugin Code tab
    4. 
              
  4. Click Run — frames, auto layout, colors, and text styles will be created
    5.
